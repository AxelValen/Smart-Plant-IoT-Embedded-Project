El proyecto

Cuidado inteligente para plantas de maceta

Smart-Plant es un sistema de monitoreo para plantas de maceta en hogares y oficinas. Mediante un ESP32 y sensores de suelo, registra en tiempo real la temperatura, la humedad y los niveles de nitrógeno (N), fósforo (P) y potasio (K).

La plataforma compara cada lectura con una amplia base de datos que contiene los rangos óptimos de crecimiento de cada especie. De esta manera, puede indicar la condición actual de la planta, generar alertas y recomendar al usuario qué acción debe realizar para mejorar su cuidado.

El ESP32 obtiene las lecturas de los sensores y las envía mediante WiFi al servidor. La plataforma web recibe la información en tiempo real, la compara con los valores óptimos almacenados para la especie seleccionada y muestra el estado de la planta, sus alertas, recomendaciones y el historial de humedad, temperatura y nutrientes NPK.

Componentes del sistema

Hardware y software

💧

Sensor de humedad

Adafruit Seesaw I²C. Mide humedad capacitiva y temperatura con corrección térmica automática a 25 °C de referencia.

🧪

Sensor NPK

Sensor RS485 Modbus RTU. Lee nitrógeno, fósforo y potasio en la tierra, los cuales son necesarios para el crecimiento de la planta.

🚿

Bomba peristáltica

Solo se enciende cuando el usuario pulsa el botón de riego después de recibir una alerta de humedad baja, y se apaga automáticamente al alcanzar el umbral correcto.

⚙️

ESP32 + servidor

El ESP32 procesa las lecturas y las transmite por WiFi al servidor, que almacena, compara y presenta los datos en la plataforma web.

Riego remoto asistido

El usuario decide cuándo regar

Cuando la humedad cae por debajo del umbral definido para la planta, el sitio web genera una alerta y habilita la acción de riego. La bomba no se enciende de manera automática: el usuario debe activarla mediante el botón correspondiente, incluso si se encuentra fuera del hogar u oficina.

Una vez iniciado el riego, el sistema continúa midiendo la humedad y apaga automáticamente la bomba cuando se alcanza el umbral adecuado. Así se evita el exceso de agua y se mantiene el control final en manos del usuario.

Historial y recomendaciones

Información útil para cada planta

La plataforma conserva el historial de humedad, temperatura y nutrientes NPK de cada planta añadida al jardín virtual. Estos datos permiten observar tendencias, identificar cambios en el suelo y consultar recomendaciones específicas según la especie.

Smart-Plant está pensado para personas que poseen varias plantas y desean cuidarlas con mayor facilidad. Mientras tengan acceso a Internet, podrán revisar su estado, recibir alertas y activar el riego desde cualquier lugar, sin tener que preocuparse constantemente por recordar cuándo echarles agua.
